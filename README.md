## Prerequisites

- Hugo Extended >= 0.110
- NodeJS + Yarn/NPM

## Setup

1. Clone the repository
```
git clone git@github.com/sumesolutions/website.gi
```

2. Download the dependencies

```
yarn install
```

3. Run the server
```
yarn start
```
